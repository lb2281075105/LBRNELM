> ## Run app(第一次运行此项目)

```
 第一步：git clone https://github.com/lb2281075105/LBRNELM.git
 第二步：cd LBRNELM && npm install
 第三步：react-native link
 第四步：react-native run-ios

```

> 理论上兼容iOS / Android，但是没在Android真机上试过，只是在模拟器上面运行了下。
